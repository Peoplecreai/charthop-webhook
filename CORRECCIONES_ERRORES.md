## Error 3: jq error con Runn API ‚ùå REQUIERE CORRECCI√ìN

```bash

     -H "Accept-Version: $RUNN_ACCEPT_VERSION" \
     "$RUNN_BASE_URL/people" \

     -H "Accept-Version: $RUNN_ACCEPT_VERSION" \
     "$RUNN_BASE_URL/people" \

  .[]

```
